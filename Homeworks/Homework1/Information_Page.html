Binary Operators



The four main binary operators are:

  • AND
    • The AND operations only equals true when all the inputs are true, and is otherwise equal to false.
  • OR
    • The OR operation only equals to true when at least one input equals to true, and is otherwise equal to false.
  • NOT
    • The NOT operations simply inverts the value of the input, which means if the input is equal to true, the output is equal to false and vice-versa.
  • XOR
    • The XOR operation is only true when the inputs have distinct values, and is otherwise(when all inputs are the same value) false.
